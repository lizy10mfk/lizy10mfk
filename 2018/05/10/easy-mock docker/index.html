通过docker部署easy-mock到内网

5月 10, 2018 503

easy-mock是一个mock数据系统，github。

需要将此系统部署至内网，供前端使用。内网服务使用了docker，所以需要为此系统配置一个docker。

easy-mock docker

easy-mock

easy-mock是一个mock数据系统，github

需要将此系统部署至内网，供前端使用。内网服务使用了docker，所以需要为此系统配置一个docker。

docker

安装

macOS

或者至官网下载安装包。

安装镜像

国内安装需要配置代理，否则速度比较差

easy-mock使用了redis(>= v4.0)、MongoDB(>= v3.4)

安装成功之后，编写Dockerfile文件及编写docker-compose.yml文件。参考文章docker nodejs

启动docker

Dockerfile和docker-compose.yml编写完成后，可以执行docker-compose up --build启动docker build

docker easy-mock
